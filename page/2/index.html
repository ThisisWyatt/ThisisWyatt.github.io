1.2 Docker镜像操作

  • 搜索镜像 docker search keyword

  • 获取镜像 docker pull imageName

  • 查看镜像信息 docker iamges

    • REPOSITORY：镜像所属的仓库

    • TAG：镜像的标签信息，用于标记同一仓库的不同镜像

      为了方便使用镜像，可以使用docker tag 名称:[标签] 新名称:[新标签]

    • IMAGE ID：镜像的唯一ID号

    可通过ID号来获取对应镜像信息：docker inspect id
