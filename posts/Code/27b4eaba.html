  • InnoDBжҳҜдёҖдёӘеҸҜйқ зҡ„дәӢеҠЎеӨ„зҗҶеј•ж“ҺпјҲеҸӮи§Ғз¬¬26з« пјүпјҢе®ғдёҚж”ҜжҢҒе…Ёж–Үжң¬жҗңзҙў
  • MEMORYеңЁеҠҹиғҪзӯүеҗҢдәҺMyISAMпјҢдҪҶз”ұдәҺж•°жҚ®еӯҳеӮЁеңЁеҶ…еӯҳпјҲдёҚжҳҜзЈҒзӣҳпјү дёӯпјҢйҖҹеәҰеҫҲеҝ«пјҲзү№еҲ«йҖӮеҗҲдәҺдёҙж—¶иЎЁпјү
  • MyISAMжҳҜдёҖдёӘжҖ§иғҪжһҒй«ҳзҡ„еј•ж“ҺпјҢе®ғж”ҜжҢҒе…Ёж–Үжң¬жҗңзҙўпјҲеҸӮи§Ғз¬¬18з« пјүпјҢ дҪҶдёҚж”ҜжҢҒдәӢеҠЎеӨ„зҗҶгҖӮ

жӣҙж–°иЎЁ

дёәжӣҙж–°иЎЁе®ҡд№үпјҢеҸҜдҪҝз”ЁALTER TABLEиҜӯеҸҘгҖӮдҪҶжҳҜпјҢзҗҶжғізҠ¶жҖҒдёӢпјҢеҪ“иЎЁдёӯеӯҳеӮЁж•°жҚ®д»ҘеҗҺпјҢиҜҘиЎЁе°ұдёҚеә”иҜҘеҶҚиў«жӣҙж–°гҖӮеңЁиЎЁзҡ„и®ҫи®ЎиҝҮзЁӢдёӯйңҖиҰҒиҠұиҙ№еӨ§йҮҸж—¶й—ҙжқҘиҖғиҷ‘пјҢд»ҘдҫҝеҗҺжңҹдёҚеҜ№иҜҘиЎЁиҝӣиЎҢеӨ§зҡ„ж”№еҠЁгҖӮ

еӨҚжқӮзҡ„иЎЁз»“жһ„жӣҙж”№дёҖиҲ¬йңҖиҰҒжүӢеҠЁеҲ йҷӨиҝҮзЁӢпјҢе®ғж¶үеҸҠд»ҘдёӢжӯҘйӘӨпјҡ

  • з”Ёж–°зҡ„еҲ—еёғеұҖеҲӣе»әдёҖдёӘж–°иЎЁ
  • дҪҝз”ЁINSERT SELECTиҜӯеҸҘд»Һж—§иЎЁеӨҚеҲ¶ж•°жҚ®еҲ°ж–°иЎЁ
  • жЈҖйӘҢеҢ…еҗ«жүҖйңҖж•°жҚ®зҡ„ж–°иЎЁ
  • йҮҚе‘ҪеҗҚж—§иЎЁпјҲеҰӮжһңзЎ®е®ҡпјҢеҸҜд»ҘеҲ йҷӨе®ғпјү
  • з”Ёж—§иЎЁеҺҹжқҘзҡ„еҗҚеӯ—йҮҚе‘ҪеҗҚж–°иЎЁ
  • ж №жҚ®йңҖиҰҒпјҢйҮҚж–°еҲӣе»әи§ҰеҸ‘еҷЁгҖҒеӯҳеӮЁиҝҮзЁӢгҖҒзҙўеј•е’ҢеӨ–й”®

и§Ҷеӣҫ

и§ҶеӣҫжҳҜиҷҡжӢҹзҡ„иЎЁгҖӮдёҺеҢ…еҗ«ж•°жҚ®зҡ„иЎЁдёҚдёҖж ·пјҢи§ҶеӣҫеҸӘеҢ…еҗ«дҪҝз”Ёж—¶еҠЁжҖҒжЈҖзҙўж•°жҚ®зҡ„жҹҘиҜўгҖӮ

е®ғжң¬иә«дёҚеҢ…еҗ«иЎЁдёӯеә”иҜҘжңүзҡ„д»»дҪ•еҲ—жҲ–ж•°жҚ®пјҢе®ғеҢ…еҗ«зҡ„жҳҜдёҖдёӘSQLжҹҘиҜў

и§Ҷеӣҫзҡ„еә”з”Ё

  • йҮҚз”ЁSQLиҜӯеҸҘ
  • з®ҖеҢ–еӨҚжқӮзҡ„SQLж“ҚдҪңгҖӮеңЁзј–еҶҷжҹҘиҜўеҗҺпјҢеҸҜд»Ҙж–№дҫҝең°йҮҚз”Ёе®ғиҖҢдёҚеҝ…зҹҘйҒ“е®ғзҡ„еҹәжң¬жҹҘиҜўз»ҶиҠӮ
  • дҪҝз”ЁиЎЁзҡ„з»„жҲҗйғЁеҲҶиҖҢдёҚжҳҜж•ҙдёӘиЎЁгҖӮ
  • дҝқжҠӨж•°жҚ®гҖӮеҸҜд»Ҙз»ҷз”ЁжҲ·жҺҲдәҲиЎЁзҡ„зү№е®ҡйғЁеҲҶзҡ„и®ҝй—®жқғйҷҗиҖҢдёҚжҳҜж•ҙдёӘ иЎЁзҡ„и®ҝй—®жқғйҷҗ
  • жӣҙж”№ж•°жҚ®ж јејҸе’ҢиЎЁзӨәгҖӮи§ҶеӣҫеҸҜиҝ”еӣһдёҺеә•еұӮиЎЁзҡ„иЎЁзӨәе’Ңж јејҸдёҚеҗҢзҡ„ж•°жҚ®

еңЁи§ҶеӣҫеҲӣе»әд№ӢеҗҺпјҢеҸҜд»Ҙз”ЁдёҺиЎЁеҹәжң¬зӣёеҗҢзҡ„ж–№ејҸеҲ©з”Ёе®ғд»¬гҖӮеҸҜд»ҘеҜ№и§Ҷеӣҫжү§иЎҢSELECTж“ҚдҪңпјҢиҝҮж»Өе’ҢжҺ’еәҸж•°жҚ®пјҢе°Ҷи§ҶеӣҫиҒ”з»“еҲ°е…¶д»–и§ҶеӣҫжҲ–иЎЁпјҢз”ҡиҮіиғҪж·»еҠ е’Ңжӣҙж–°ж•°жҚ®пјҲж·»еҠ е’Ңжӣҙж–°ж•°жҚ®еӯҳеңЁжҹҗдәӣйҷҗеҲ¶гҖӮе…ідәҺиҝҷдёӘеҶ…е®№зЁҚ еҗҺиҝҳиҰҒеҒҡиҝӣдёҖжӯҘзҡ„д»Ӣз»ҚпјүгҖӮ

еҰӮжһңдҪ з”ЁеӨҡдёӘиҒ”з»“е’ҢиҝҮж»ӨеҲӣе»әдәҶеӨҚжқӮзҡ„и§ҶеӣҫжҲ–иҖ…еөҢеҘ—дәҶи§ҶеӣҫпјҢеҸҜиғҪдјҡеҸ‘зҺ°жҖ§иғҪдёӢйҷҚеҫ—еҫҲеҺүе®ігҖӮ

и§Ҷеӣҫзҡ„и§„еҲҷе’ҢйҷҗеҲ¶

  • и§Ҷеӣҫеҝ…йЎ»е”ҜдёҖе‘ҪеҗҚ
  • еҲӣе»әзҡ„и§Ҷеӣҫж•°зӣ®жІЎжңүйҷҗеҲ¶
  • еҝ…йЎ»е…·жңүи¶іеӨҹзҡ„и®ҝй—®жқғйҷҗ
  • и§ҶеӣҫеҸҜд»ҘеөҢеҘ—пјҢеҚіеҸҜд»ҘеҲ©з”Ёд»Һе…¶д»–и§ҶеӣҫдёӯжЈҖзҙўж•°жҚ®зҡ„жҹҘиҜўжқҘжһ„йҖ  дёҖдёӘи§Ҷеӣҫ
  • ORDER BYеҸҜд»Ҙз”ЁеңЁи§ҶеӣҫдёӯпјҢдҪҶеҰӮжһңд»ҺиҜҘи§ҶеӣҫжЈҖзҙўж•°жҚ®SELECTдёӯд№ҹеҗ«жңүORDER BYпјҢйӮЈд№ҲиҜҘи§Ҷеӣҫдёӯзҡ„ORDER BYе°Ҷиў«иҰҶзӣ–
  • и§ҶеӣҫдёҚиғҪзҙўеј•пјҢд№ҹдёҚиғҪжңүе…іиҒ”зҡ„и§ҰеҸ‘еҷЁжҲ–й»ҳи®ӨеҖј
  • и§ҶеӣҫеҸҜд»Ҙе’ҢиЎЁдёҖиө·дҪҝз”ЁгҖӮдҫӢеҰӮпјҢзј–еҶҷдёҖжқЎиҒ”з»“иЎЁе’Ңи§Ҷеӣҫзҡ„SELECT иҜӯеҸҘгҖӮ

дҪҝз”Ёи§Ҷеӣҫ

  • CREATE VIEWиҜӯеҸҘжқҘеҲӣе»ә
  • SHOW CREATE VIEW viewnameпјӣжқҘжҹҘзңӢеҲӣе»әи§Ҷеӣҫзҡ„иҜӯ
  • з”ЁDROPеҲ йҷӨи§ҶеӣҫпјҢе…¶иҜӯжі•дёәDROP VIEW viewname
  • жӣҙж–°и§Ҷеӣҫж—¶пјҢеҸҜд»Ҙе…Ҳз”ЁDROPеҶҚз”ЁCREATEпјҢд№ҹеҸҜд»ҘзӣҙжҺҘз”ЁCREATE OR REPLACE VIEWгҖӮеҰӮжһңиҰҒжӣҙж–°зҡ„и§ҶеӣҫдёҚеӯҳеңЁпјҢеҲҷз¬¬2жқЎжӣҙж–°иҜӯеҸҘдјҡеҲӣ е»әдёҖдёӘи§ҶеӣҫпјӣеҰӮжһңиҰҒжӣҙж–°зҡ„и§ҶеӣҫеӯҳеңЁпјҢеҲҷз¬¬2жқЎжӣҙж–°иҜӯеҸҘдјҡжӣҝжҚўеҺҹ жңүи§Ҷеӣҫ

жӣҙж–°и§Ҷеӣҫ

йҖҡеёёпјҢи§ҶеӣҫжҳҜеҸҜжӣҙж–°зҡ„пјҢдҪҶжҳҜпјҢе№¶йқһжүҖжңүи§ҶеӣҫйғҪжҳҜеҸҜжӣҙж–°зҡ„гҖӮеҹәжң¬дёҠеҸҜд»ҘиҜҙпјҢеҰӮжһңMySQLдёҚиғҪжӯЈзЎ®ең°зЎ®е®ҡиў«жӣҙж–°зҡ„еҹәж•°жҚ®пјҢеҲҷдёҚе…Ғи®ёжӣҙж–°пјҲеҢ…жӢ¬жҸ’е…Ҙе’ҢеҲ йҷӨпјүгҖӮиҝҷе®һйҷ…дёҠж„Ҹе‘ізқҖпјҢеҰӮжһңи§Ҷеӣҫе®ҡд№үдёӯжңүд»ҘдёӢж“ҚдҪңпјҢеҲҷдёҚиғҪиҝӣиЎҢи§Ҷеӣҫзҡ„жӣҙж–°пјҡ

  • еҲҶз»„
  • иҒ”з»“

еӯҳеӮЁиҝҮзЁӢ

иҝ„д»ҠдёәжӯўпјҢдҪҝз”Ёзҡ„еӨ§еӨҡж•°SQLиҜӯеҸҘйғҪжҳҜй’ҲеҜ№дёҖдёӘжҲ–еӨҡдёӘиЎЁзҡ„еҚ•жқЎиҜӯ еҸҘгҖӮе№¶йқһжүҖжңүж“ҚдҪңйғҪиҝҷд№Ҳз®ҖеҚ•пјҢз»ҸеёёдјҡжңүдёҖдёӘе®Ңж•ҙзҡ„ж“ҚдҪңйңҖиҰҒеӨҡжқЎиҜӯеҸҘ жүҚиғҪе®ҢжҲҗгҖӮдҫӢеҰӮпјҢиҖғиҷ‘д»ҘдёӢзҡ„жғ…еҪў

  • дёәдәҶеӨ„зҗҶи®ўеҚ•пјҢйңҖиҰҒж ёеҜ№д»ҘдҝқиҜҒеә“еӯҳдёӯжңүзӣёеә”зҡ„зү©е“Ғ
  • еҰӮжһңеә“еӯҳжңүзү©е“ҒпјҢиҝҷдәӣзү©е“ҒйңҖиҰҒйў„е®ҡд»ҘдҫҝдёҚе°Ҷе®ғд»¬еҶҚеҚ–з»ҷеҲ«зҡ„дәәпјҢе№¶дё”иҰҒеҮҸе°‘еҸҜз”Ёзҡ„зү©е“Ғж•°йҮҸд»ҘеҸҚжҳ жӯЈзЎ®зҡ„еә“еӯҳйҮҸ
  • еә“еӯҳдёӯжІЎжңүзҡ„зү©е“ҒйңҖиҰҒи®ўиҙӯпјҢиҝҷйңҖиҰҒдёҺдҫӣеә”е•ҶиҝӣиЎҢжҹҗз§ҚдәӨдә’
  • е…ідәҺе“Әдәӣзү©е“Ғе…Ҙеә“пјҲе№¶дё”еҸҜд»Ҙз«ӢеҚіеҸ‘иҙ§пјүе’Ңе“Әдәӣзү©е“ҒйҖҖи®ўпјҢйңҖиҰҒйҖҡзҹҘзӣёеә”зҡ„е®ўжҲ·

жү§иЎҢиҝҷдёӘеӨ„зҗҶйңҖиҰҒй’ҲеҜ№и®ёеӨҡиЎЁзҡ„еӨҡжқЎMySQLиҜӯеҸҘгҖӮжӯӨеӨ–пјҢйңҖиҰҒжү§иЎҢзҡ„е…·дҪ“иҜӯеҸҘеҸҠе…¶ж¬ЎеәҸд№ҹдёҚжҳҜеӣәе®ҡзҡ„пјҢе®ғд»¬еҸҜиғҪдјҡпјҲе’Ңе°Ҷпјүж №жҚ®е“Әдәӣзү©е“ҒеңЁеә“еӯҳдёӯе“ӘдәӣдёҚеңЁиҖҢеҸҳеҢ–гҖӮ

еӯҳеӮЁиҝҮзЁӢз®ҖеҚ•жқҘиҜҙпјҢе°ұжҳҜдёәд»ҘеҗҺзҡ„дҪҝз”ЁиҖҢдҝқеӯҳзҡ„дёҖжқЎжҲ–еӨҡжқЎMySQLиҜӯеҸҘзҡ„йӣҶеҗҲгҖӮ

иҝӣдёҖжӯҘд»Ӣз»Қ

  • еә”иҜҘз”Ёи§ҰеҸ‘еҷЁжқҘдҝқиҜҒж•°жҚ®зҡ„дёҖиҮҙжҖ§пјҲеӨ§е°ҸеҶҷгҖҒж јејҸзӯүпјүгҖӮеңЁи§ҰеҸ‘еҷЁдёӯжү§иЎҢиҝҷз§Қзұ»еһӢзҡ„еӨ„зҗҶзҡ„дјҳзӮ№жҳҜе®ғжҖ»жҳҜиҝӣиЎҢиҝҷз§ҚеӨ„зҗҶпјҢиҖҢдё”жҳҜйҖҸжҳҺең°иҝӣиЎҢпјҢдёҺе®ўжҲ·жңәеә”з”Ёж— е…ігҖӮ

  • йҒ—жҶҫзҡ„жҳҜпјҢMySQLи§ҰеҸ‘еҷЁдёӯдёҚж”ҜжҢҒCALLиҜӯеҸҘгҖӮиҝҷиЎЁзӨәдёҚиғҪд»Һи§ҰеҸ‘еҷЁеҶ…и°ғз”ЁеӯҳеӮЁиҝҮзЁӢгҖӮжүҖйңҖзҡ„еӯҳеӮЁиҝҮзЁӢд»Јз ҒйңҖиҰҒеӨҚеҲ¶еҲ°и§ҰеҸ‘еҷЁеҶ…гҖӮ





дәӢеҠЎеӨ„зҗҶ

дәӢеҠЎеӨ„зҗҶпјҲtransaction processingпјүеҸҜд»Ҙз”ЁжқҘз»ҙжҠӨж•°жҚ®еә“зҡ„е®Ңж•ҙжҖ§пјҢе®ғ дҝқиҜҒжҲҗжү№зҡ„MySQLж“ҚдҪңиҰҒд№Ҳе®Ңе…Ёжү§иЎҢпјҢиҰҒд№Ҳе®Ңе…ЁдёҚжү§иЎҢгҖӮ

еҲ©з”ЁдәӢеҠЎеӨ„зҗҶпјҢеҸҜд»ҘдҝқиҜҒдёҖз»„ж“ҚдҪңдёҚдјҡдёӯйҖ”еҒңжӯўпјҢе®ғд»¬жҲ–иҖ…дҪңдёәж•ҙдҪ“жү§иЎҢпјҢжҲ–иҖ…е®Ңе…ЁдёҚжү§иЎҢпјҲйҷӨйқһжҳҺзЎ®жҢҮзӨәпјүгҖӮеҰӮжһңжІЎжңүй”ҷиҜҜеҸ‘з”ҹпјҢж•ҙз»„иҜӯеҸҘжҸҗдәӨз»ҷпјҲеҶҷеҲ°пјүж•°жҚ®еә“иЎЁгҖӮеҰӮжһңеҸ‘з”ҹй”ҷиҜҜпјҢеҲҷиҝӣиЎҢеӣһйҖҖпјҲж’Өй”Җпјүд»ҘжҒўеӨҚж•°жҚ®еә“еҲ°жҹҗдёӘе·ІзҹҘдё”е®үе…Ёзҡ„зҠ¶жҖҒгҖӮ

  • дәӢеҠЎпјҲtransactionпјүжҢҮдёҖз»„SQLиҜӯеҸҘпјӣ
  • еӣһйҖҖпјҲrollbackпјүжҢҮж’Өй”ҖжҢҮе®ҡSQLиҜӯеҸҘзҡ„иҝҮзЁӢпјӣ
  • жҸҗдәӨпјҲcommitпјүжҢҮе°ҶжңӘеӯҳеӮЁзҡ„SQLиҜӯеҸҘз»“жһңеҶҷе…Ҙж•°жҚ®еә“иЎЁпјӣ
  • дҝқз•ҷзӮ№пјҲsavepointпјүжҢҮдәӢеҠЎеӨ„зҗҶдёӯи®ҫзҪ®зҡ„дёҙж—¶еҚ дҪҚз¬ҰпјҲplace- holderпјүпјҢдҪ еҸҜд»ҘеҜ№е®ғеҸ‘еёғеӣһйҖҖпјҲдёҺеӣһйҖҖж•ҙдёӘдәӢеҠЎеӨ„зҗҶдёҚеҗҢ

дҪҝз”ЁROLLBACK

MySQLзҡ„ROLLBACKе‘Ҫд»Өз”ЁжқҘеӣһйҖҖпјҲж’Өй”ҖпјүMySQLиҜӯеҸҘгҖӮ

йҰ–е…Ҳжү§иЎҢдёҖжқЎSELECTд»ҘжҳҫзӨәиҜҘиЎЁдёҚдёәз©әгҖӮз„¶еҗҺејҖе§ӢдёҖдёӘдәӢеҠЎеӨ„зҗҶпјҢз”ЁдёҖжқЎDELETEиҜӯеҸҘеҲ йҷӨordertotalsдёӯзҡ„жүҖжңүиЎҢгҖӮеҸҰдёҖжқЎ SELECTиҜӯеҸҘйӘҢиҜҒordertotalsзЎ®е®һдёәз©әгҖӮиҝҷж—¶з”ЁдёҖжқЎROLLBACKиҜӯеҸҘеӣһйҖҖ START TRANSACTIONд№ӢеҗҺзҡ„жүҖжңүиҜӯеҸҘпјҢжңҖеҗҺдёҖжқЎSELECTиҜӯеҸҘжҳҫзӨәиҜҘиЎЁдёҚдёәз©әгҖӮ

дҪҝз”ЁCOMMIT

дёҖиҲ¬зҡ„MySQLиҜӯеҸҘйғҪжҳҜзӣҙжҺҘй’ҲеҜ№ж•°жҚ®еә“иЎЁжү§иЎҢе’Ңзј–еҶҷзҡ„гҖӮиҝҷе°ұжҳҜжүҖи°“зҡ„йҡҗеҗ«жҸҗдәӨпјҲimplicit commitпјүпјҢеҚіжҸҗдәӨпјҲеҶҷжҲ–дҝқеӯҳпјүж“ҚдҪңжҳҜиҮӘеҠЁиҝӣиЎҢзҡ„гҖӮ

дҪҶжҳҜпјҢеңЁдәӢеҠЎеӨ„зҗҶеқ—дёӯпјҢжҸҗдәӨдёҚдјҡйҡҗеҗ«ең°иҝӣиЎҢгҖӮдёәиҝӣиЎҢжҳҺзЎ®зҡ„жҸҗдәӨпјҢдҪҝз”ЁCOMMITиҜӯеҸҘпјҢеҰӮдёӢжүҖзӨәпјҡ

д»Һзі»з»ҹдёӯе®Ңе…ЁеҲ йҷӨи®ўеҚ•20010гҖӮеӣ дёәж¶үеҸҠжӣҙж–°дёӨдёӘж•°жҚ®еә“иЎЁordersе’ҢorderItemsпјҢжүҖд»ҘдҪҝз”ЁдәӢеҠЎеӨ„зҗҶеқ—жқҘдҝқиҜҒи®ўеҚ•дёҚиў«йғЁеҲҶеҲ йҷӨгҖӮжңҖеҗҺзҡ„COMMITиҜӯеҸҘд»…еңЁдёҚеҮәй”ҷж—¶еҶҷеҮәжӣҙж”№гҖӮеҰӮ жһңз¬¬дёҖжқЎDELETEиө·дҪңз”ЁпјҢдҪҶз¬¬дәҢжқЎеӨұиҙҘпјҢеҲҷDELETEдёҚдјҡжҸҗдәӨпјҲе®һйҷ…дёҠпјҢе®ғжҳҜиў«иҮӘеҠЁж’Өй”Җзҡ„пјү

йҡҗеҗ«дәӢеҠЎе…ій—ӯ еҪ“COMMITжҲ–ROLLBACKиҜӯеҸҘжү§иЎҢеҗҺпјҢдәӢеҠЎдјҡиҮӘ еҠЁе…ій—ӯпјҲе°ҶжқҘзҡ„жӣҙж”№дјҡйҡҗеҗ«жҸҗдәӨпјүгҖӮ

дҪҝз”Ёдҝқз”ЁзӮ№

з®ҖеҚ•зҡ„ROLLBACKе’ҢCOMMITиҜӯеҸҘе°ұеҸҜд»ҘеҶҷе…ҘжҲ–ж’Өй”Җж•ҙдёӘдәӢеҠЎеӨ„зҗҶгҖӮдҪҶ жҳҜпјҢеҸӘжҳҜеҜ№з®ҖеҚ•зҡ„дәӢеҠЎеӨ„зҗҶжүҚиғҪиҝҷж ·еҒҡпјҢжӣҙеӨҚжқӮзҡ„дәӢеҠЎеӨ„зҗҶеҸҜиғҪйңҖиҰҒйғЁеҲҶжҸҗдәӨжҲ–еӣһйҖҖгҖӮ

е…ЁзҗғеҢ–е’Ңжң¬ең°еҢ–

  • зј–з ҒпјҡжҹҗдёӘеӯ—з¬ҰйӣҶжҲҗе‘ҳзҡ„еҶ…йғЁиЎЁзӨәпјӣ
